# 3.2.1 - Bash Reserved Words

和多数的编程语言一样 Bash 也有自己的 reserved words(keywords)，通常被用在 compound commands 开头或者是结尾

一共有如下几个 reversed word

- if
- then
- elfi
- else
- fi
- time
- for
- in
- until
- while
- do
- done
- {
- }
- [[
- ]]
- !
