# window 命令

## 快捷命令

- `win + E`

  打开资源管理器

- `win + I`

  打开windows设置

- `win + shift + S`

  框选截屏

- `win + D`

  显示桌面

- `win + L`

  锁屏

- `ctrl + shift + esc`

  任务管理器

## 系统组件

- `control`

  ==控制面板==

- `msinfo32`

  ==查看系统详细信息==

- `systeminfo`

  在Dos中使用

- `diskmgmt.msc`

  磁盘管理(用我的电脑右击管理, 来做磁盘分区存在问题)

- `taskmgr`

  任务管理器

- `services.msc`

  服务列表

- `cleanmgr`

  垃圾整理

- `regedit`

  注册列表

- `mspaint`

  画图工具

- `calc`

  计算器

- `winver`

  windows系统版本

- `dxdiag`

  检查DirectX信息

- `msconfig`

  查看系统引导
