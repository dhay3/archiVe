# hugo - Overview

hugo 是一个静态站点生成工具，和 Hexo 类似
在 hugo 中有几个主要的对象

1. homepage

   对应 `index.html`，访问 `example.org` 或者 `example.org/index.html` 渲染对内容

2. page

   对应 `content`目录下的文章，访问 `example.org`

3. section/list

   对应 `content` 目录下的目录
