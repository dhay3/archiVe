# hugo - Taxonomies

taxonomies 指的是分类
Hugo 中有默认的 taxonomies ( `tags`，`categories` )，无需在 site configuratoin 中指定也可以使用 hugo 的分类功能
例如，`content/post1.md` 中的 front matter 如下
```
---
categories:
- Development
- Python
- Crawler
---
```
`content/post2.md` 中的 front matter 如下
```
---
categories:
- Development
- Java
- Web
---
```
