# Typora Shortcuts

ctrl + n 新建

ctrl + s 保存

ctrl + f 查找替换

ctrl + / 源码模式

ctrl + shift + l 左边导航栏

ctrl + shift + i 插入图片

ctrl + shift + k 代码块

ctrl + shift + m 数学公式块

ctrl + b 加粗

ctrl + i 斜体

ctrl + u 下划线

ctrl + k 插入超链接

ctrl + t 插入表格

ctrl + \ 清楚 Markdown 格式

F8 专注模式

F9 打字机模式

Shift + F12 开发者工具
