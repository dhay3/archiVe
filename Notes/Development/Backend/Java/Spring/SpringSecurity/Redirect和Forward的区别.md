# Redirect和Forward的区别

参考:

https://blog.csdn.net/u010452388/article/details/80398929

|                    | redirect/重定向 | forward/请求转发          |
| ------------------ | --------------- | ------------------------- |
| 数据共享           | 不能共享数据    | 可以共享request里面的数据 |
| 地址栏             | 地址栏改变      | 地址栏不变                |
| 效率               | 低              | 高                        |
| 请求次数           | 两次            | 一次                      |
| 可以跳转到什么资源 | 任意资源        | 项目内部                  |

所以重定向后的请求同样会被filter拦截, 但是请求转发不会被拦截,
